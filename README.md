# Cognify AI - Learn Smarter, Not Harder

Cognify is a premium, AI-powered learning ecosystem designed to revolutionize how students interact with educational content. By leveraging advanced Generative AI (Google Gemini), Cognify provides personalized tutoring, instant assessment, and data-driven insights.

## 🚀 Problem Statement
Traditional education often lacks personalization, leaving students overwhelmed by vast amounts of information without a clear path to mastery. Static study materials and delayed feedback make it difficult for learners to identify their weaknesses and track progress effectively in real-time.

**Key Challenges addressed:**
- **Information Overload:** Students struggle to synthesize structured notes from raw content.
- **Generic Assessments:** Lack of immediate, personalized practice material.
- **Limited Access to Help:** No real-time guidance for complex doubts outside classrooms.
- **Vague Progress Tracking:** Difficulty in identifying specific "Weak Areas."

## ✨ Core Features
- **📝 AI Notes Generator:** Instantly converts any topic into structured study notes (Supports English & Hindi).
- **🧠 Adaptive Quiz Engine:** Generates personalized quizzes to test mastery with instant feedback.
- **🤖 24/7 AI Tutor:** A persistent, context-aware assistant to answer doubts and explain complex concepts.
- **📊 Intelligent Analytics:** High-fidelity dashboard with Skill Radars, Activity Heatmaps, and Trend Analysis.
- **👨‍🏫 Teacher Mode:** Advanced tools for educators to generate content and monitor student performance.
- **🏆 Gamified Learning:** Experience Points (XP) and Badges system to keep students engaged.

## 🛠 Tech Stack
- **Frontend:** React.js, Tailwind CSS, Framer Motion, Three.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI:** Google Gemini API
- **Auth:** Firebase Authentication

## 📦 Setup Instructions
1. Clone the repository
2. Install dependencies:
   ```bash
   # Root folder
   npm install
   # Server folder
   cd server && npm install
   ```
3. Set up your `.env` files (refer to `.env.example`)
4. Run the development servers:
   ```bash
   # Frontend
   npm run dev
   # Backend
   cd server && npm run dev
   ```
